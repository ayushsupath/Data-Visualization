# 📊 Data-Visualization

A practice repository exploring data visualization techniques in Python — from basic plots to distribution analysis — using **Matplotlib**, **Seaborn**, and **Plotly**. Built to strengthen EDA (Exploratory Data Analysis) and data storytelling skills.

## 🚀 What's Inside

| File | Description |
|------|-------------|
| `MatplotLibs.ipynb` | Fundamentals of plotting with Matplotlib — line charts, bar charts, scatter plots, subplots, and styling |
| `DistributionPlots.ipynb` | Distribution analysis using histograms, KDE plots, box plots, and violin plots |
| `BasicPlot.png` | Sample output visualization generated from the notebooks |

## 🛠️ Tech Stack

- **Python 3**
- **Matplotlib** — core plotting library
- **Seaborn** — statistical data visualization
- **Plotly** — interactive charts and dashboards
- **Jupyter Notebook** — for interactive development

## 📈 Topics Covered

- Basic chart types (line, bar, scatter, pie)
- Customizing plots (titles, labels, legends, colors, styles)
- Subplots and multi-panel figures
- Distribution plots (histogram, KDE, box plot, violin plot)
- Statistical visualization with Seaborn
- Interactive visualizations with Plotly

## 🔧 Getting Started

Clone the repo:
```bash
git clone https://github.com/ayushsupath/Data-Visualization.git
cd Data-Visualization
```

Install dependencies:
```bash
pip install matplotlib seaborn plotly pandas numpy jupyter
```

Run the notebooks:
```bash
jupyter notebook
```

## 🎯 Purpose

This repo is a hands-on learning space for mastering data visualization concepts — an essential skill for data analysis, EDA, and building insight-driven dashboards.
